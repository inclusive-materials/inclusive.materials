⚙️ One-Time Setup — OAuth Client ID

You need a free Google Cloud OAuth Client ID to allow this dashboard to read your analytics securely. This only needs to be done once.

✓ Saved!

Step-by-step instructions to get your Client ID:

  1. Go to console.cloud.google.com — sign in with your Google account
  2. Click the project dropdown at the top → New Project → name it Inclusive Materials → Create
  3. In the left menu go to APIs & Services → Library
  4. Search for Google Analytics Data API → click it → click Enable
  5. Go to APIs & Services → OAuth consent screen → choose External → Create
  6. Fill in App name (Inclusive Materials Dashboard) and your email → Save and Continue through all steps
  7. Go to APIs & Services → Credentials → + Create Credentials → OAuth client ID
  8. Application type: Web application
  9. Under Authorized JavaScript origins, add: https://inclusivematerials.com
  10. Click Create → copy the Client ID (ends in .apps.googleusercontent.com)
  11. Paste it in the field above → click Save Client ID → then click Sign in with Google
