GitHub Token

A GitHub Personal Access Token lets this page save your content directly to your repository. It is stored only in your browser and never sent anywhere else.

How to create your token:

  1. Go to github.com → Settings → Developer Settings → Fine-grained tokens
  2. Click Generate new token
  3. Give it a name like inclusive-materials-cms
  4. Under Repository access, choose Only select repositories → pick inclusive.materials
  5. Under Permissions → Repository permissions, set Contents to Read and write
  6. Click Generate token and copy it

Token is saved in your browser's local storage. If you clear browser data, you'll need to re-enter it.

About Content Storage

📝 Blog posts are saved as .md files in docs/_data/blog/
⭐ Products are saved as .json files in docs/_data/resources/
🖼️ Photos are stored in docs/photos/ — upload new photos via GitHub directly
⚡ Changes go live within 1–2 minutes after saving (GitHub Pages rebuild)
